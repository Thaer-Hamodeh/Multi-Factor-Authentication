To get full functionality of the app you should install the following dependencies:
1- install python3
2- install pip
3- pip install flask
4- pip install flask-wtf
5- pip install flask-sqlalchemy
6- pip install flask-bcrypt
7- pip install flask-login
8- pip install flask-mail
9- Sqlite should be install
10- pip install otpauth
11- pip install qrcode
12- pip install Flask-Session
13- pip install opencv-python
14- pip install CMake
15- pip install face_recognition
